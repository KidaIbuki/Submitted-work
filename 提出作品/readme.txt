◆タイトル[ShaderProject]================================================
◇使用言語 : C++ , HLSL
◇SDK      : DirectX9
◇開発期間 : 2025/03～2025/06

◆動作確認済み環境◆
Windows11

◆操作方法◆
  [W,A,S,D]キー　　球体モデルの移動
  [F,G]キー　　カメラの左右旋回
  [N,M]キー　　球体モデルの上下移動
  [V]キー　　球体モデルのx軸回転
  [B]キー　　球体モデルのy軸回転
  [Q]キー　　回転リセット
  
◆ImGui操作方法◆
・ light Dir
  ▶ライトの位置を変えることができます
  
・WaveChange
  ▶Texture1,2,3を押すことで波シェーダーの表面を変えることができます
  ▶ Animationのスライダーで波の動きを速めたりすりことができます
  
・EffectAccumulation
  ▶OpenImGuiのチェックを外すとImGuiの欄を出せます
  ▶EffectActiveのチェックを入れるとポストエフェクトを表示できます。
  ▶EffectColorRGBの数値を変えることでポストエフェクトの色を変えれます
  ▶EffectPowerの数値を変えると円と尖りの強弱を変えることができます。
  ▶EffectSizeの数値を変えるとポストエフェクトのサイズを変えることができます・
  
・ToneAdjustのチェックを外すとImGuiの欄を出せます。
  ▶EffectActiveのチェックを入れることでポストエフェクトを表示できます
  ▶EffectColorRGBの数値を変えることでポストエフェクトの色を変えることができます。
  
  
・ESCでexeを終了することができます。
